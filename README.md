# zoxweb-core :: Enterprise Java Utility Library

[![Maven Central](https://img.shields.io/badge/Maven%20Central-2.3.8-blue)](https://mvnrepository.com/artifact/org.zoxweb/zoxweb-core)

## Overview

**zoxweb-core** is a comprehensive enterprise Java utility library by XlogistX with 670+ classes across 45+ packages. It provides robust, production-ready components for building scalable Java applications.

## Key Features

### Networking & Communications
- **NIO/TLS Socket Framework** - High-performance non-blocking I/O with full TLS/SSL support
- **WebSocket Support** - Complete WebSocket protocol implementation
- **HTTP Client/Server** - Comprehensive HTTP handling with multi-part support
- **Network Tunneling** - Channel relay and datagram support

### Security & Authentication
- **JWT/OAuth Support** - Full JWT encoding/decoding and OAuth integration
- **Cryptography** - Encryption, hashing, and secure credential management
- **Authentication Framework** - Flexible authentication handlers and token management
- **IP Filtering** - Configurable IP blocking and filtering rules

### API Development
- **RESTful API Builder** - Toolkit for building and consuming REST APIs
- **HTTP Authorization** - Basic, Bearer, and JWT authorization handlers
- **API Management** - Service providers, notifications, and transactional support

### Data & Persistence
- **DAO Abstractions** - Clean data access object patterns with CRUD operations
- **GSON Serialization** - JSON serialization utilities and type converters
- **Query Framework** - Flexible query builders and matchers

### Concurrency & Scheduling
- **Async Task Scheduling** - Thread pool management and executor services
- **Finite State Machine** - Flexible state machine implementation for complex workflows
- **Event-Driven Architecture** - Event listeners, observers, and callbacks

### Utilities
- **196+ Utility Classes** - Extensive shared utilities for common programming needs
- **I/O Utilities** - Stream processing, multi-digest hashing, byte array handling
- **Configuration System** - Flexible NVConfig-based configuration management

### Domain-Specific
- **IoT Device Management** - Device factory, protocol handling, and port management
- **Billing/Accounting** - Financial transactions, currency handling, billing accounts
- **GWT Compatible** - Shared modules designed for GWT client-server code sharing

## Installation

### Maven
```xml
<dependency>
    <groupId>org.zoxweb</groupId>
    <artifactId>zoxweb-core</artifactId>
    <version>2.3.9</version>
</dependency>
```

### Gradle
```groovy
implementation 'org.zoxweb:zoxweb-core:2.3.9'
```

## Requirements

- Java 8 or higher
- Maven 3.x (for building from source)
